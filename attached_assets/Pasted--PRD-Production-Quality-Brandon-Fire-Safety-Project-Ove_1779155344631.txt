# PRD: Production-Quality  Brandon Fire & Safety

  

## Project Overview

Build a modern, production-quality single-page local business website for a fire safety company using:

* HTML
* CSS
* Vanilla JavaScript

The website should feel:

* Professional
* Trustworthy
* Modern
* Fast
* Mobile-friendly
* Optimized for SEO
* Ready for real client deployment

The project must run immediately inside Replit with zero additional setup.

Navigation items:

* Home
* About
* Reviews
* Services
* Contact

## Smooth Scroll Navigation

When users click a navigation item:

* Smoothly scroll to the matching section
* Offset correctly for sticky navigation height

## Active Navigation Highlighting

While scrolling:

* Highlight the active navigation link
* Use JavaScript Intersection Observer or scroll detection
